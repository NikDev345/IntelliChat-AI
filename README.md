# 🚀 Overview

Gemini Memory Chatbot is a conversational AI application built using Google's Gemini API and Python.

Unlike traditional single-prompt AI programs, this chatbot maintains conversational context throughout the session, allowing users to have natural and meaningful multi-turn conversations.

It demonstrates core AI engineering concepts including:

* Prompt Management
* Chat Sessions
* Memory Handling
* Continuous User Interaction
* API Integration
* Error Handling

---

# ✨ Features

### 🧠 Conversation Memory

Remembers previous messages within the active chat session.

### 💬 Continuous Chat Loop

Chat endlessly until the user decides to exit.

### ⚡ Gemini Powered

Uses Google's powerful Gemini models for intelligent responses.

### 🚪 Smart Exit Command

Type:

```bash
quit
```

to gracefully end the conversation.

### 🐍 Pure Python

No heavy frameworks.

Simple.

Clean.

Easy to understand.

# 📦 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Gemini-Memory-Chatbot.git
```

Move into project folder

```bash
cd Gemini-Memory-Chatbot
```

Install dependencies

```bash
pip install google-genai
```

---

# 🔑 API Key Setup

Create your Gemini API Key.

Add it inside your code:

```python
client = genai.Client(
    api_key="YOUR_API_KEY"
)
```

---

# ▶️ Running the Application

```bash
python api.py
```
