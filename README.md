# 🤖 Gemini Memory Chatbot

A conversational AI chatbot built using Google's Gemini API and Python. Unlike basic single-prompt AI applications, this chatbot maintains conversation context through a persistent chat session, enabling natural multi-turn interactions.

---

## 🚀 Features

* 💬 Continuous conversation support
* 🧠 Context-aware memory
* ⚡ Powered by Google Gemini API
* 🔄 Infinite chat loop
* 🚪 Quit command support
* 🐍 Pure Python implementation
* 📚 Beginner-friendly project structure

---

## 📸 Example

```text
You: My name is Nikhil

AI: Nice to meet you, Nikhil!

You: What is my name?

AI: Your name is Nikhil.

You: quit

Goodbye!
```

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Gemini-Memory-Chatbot.git
```

Navigate into the project:

```bash
cd Gemini-Memory-Chatbot
```

Install dependencies:

```bash
pip install google-genai
```

---

## 🔑 Setup API Key

Replace the API key inside the script:

```python
client = genai.Client(api_key="YOUR_API_KEY")
```

Or use environment variables for improved security.

---

## ▶️ Run the Project

```bash
python api.py
```
